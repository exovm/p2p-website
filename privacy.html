1. introduction

welcome to p2p, a discord bot designed to provide verification, moderation, and server management tools. this privacy policy explains how we collect, use, and protect user data when you or your server use p2p. by adding or using p2p in your discord server, you agree to the terms described below.

if you have any questions, please contact us at tarajijonesm21811@gmail.com or through discord (exovm).

2. information we collect

a. user information

p2p may collect and temporarily store the following discord-related data:

  • discord user id
  • discord username and discriminator (example: username#0000)
  • server id and name
  • role assignments and permissions
  • message content (only when necessary for moderation logs or verification checks)

p2p does not collect personal information such as email addresses, passwords, ip addresses, or payment information.

b. server information

to function properly, p2p may collect:

  • server name and id
  • channel ids
  • role ids and permissions
  • server settings related to p2p's features (such as verification configurations and moderation preferences)

c. logs

p2p may temporarily log certain actions for debugging, moderation, or performance monitoring. these logs may include:

  • user command usage
  • timestamps of moderation actions (e.g., bans, mutes, verifications)
  • error reports

logs are stored securely and automatically deleted after a reasonable period (usually within 30–60 days).

3. how we use the information

the data collected by p2p is used solely to:

  • operate and maintain bot functions
  • perform verification and moderation tasks
  • improve reliability and performance
  • investigate abuse or technical issues
  • enforce the terms of service

we do not sell, trade, or share user data with third parties.

4. data retention

we keep collected data only as long as it's necessary for p2p to function. most information (like verification or moderation logs) is automatically deleted after 30–60 days. if a server removes p2p, all stored configuration data for that server is deleted automatically within 72 hours.

5. data security

we take security seriously.

  • all stored data is encrypted and accessible only to the bot owner.
  • no raw data is shared publicly.
  • access to logs and configurations is restricted to authorized personnel.

while we use reasonable security measures, no system is completely secure. by using p2p, you acknowledge and accept this risk.

6. your rights

if you are a discord user in a server using p2p, you may request:

  • a copy of any data stored about you
  • correction of inaccurate information
  • deletion of stored data

requests can be made through email (tarajijonesm21811@gmail.com) or discord (exovm). we may ask for verification to confirm your identity before fulfilling your request.

7. third-party services

p2p operates entirely within the discord api framework. therefore, your use of p2p is also governed by discord's privacy policy and terms of service. p2p does not integrate with or send data to any third-party services outside of discord.

8. children's privacy

p2p is intended for users aged 13 and older, consistent with discord's terms of service. we do not knowingly collect or store data from anyone under the age of 13. if we discover that we have received data from a minor, it will be deleted immediately.

9. changes to this policy

we may update this privacy policy from time to time to reflect changes in how p2p operates. updates will be posted publicly, and the "last updated" date will change accordingly. continued use of p2p after changes means you accept the revised policy.
